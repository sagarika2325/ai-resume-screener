# 🧠 AI-Powered Resume Screening & Candidate Insights Tool

This project uses AI and data analytics to automate the process of resume screening, candidate-job matching, and generating actionable insights for recruiters.

## 🚀 What This Project Does
- Parses resumes to extract key skills and experience
- Matches candidates to job roles using NLP and logic
- Builds dashboards to visualize candidate pools
- Uses AI tools to generate summaries and insights

## 🛠️ Tools & Tech Stack
- Python (Pandas, Scikit-learn, NLTK)
- SQL & Excel
- Tableau / Power BI
- OpenAI API (for insights generation)
- Git & GitHub

## 📁 Project Structure
- `/data`: Resume & job description CSV files  
- `/notebooks`: Jupyter Notebooks for each phase (cleaning, analysis, NLP)  
- `/visualizations`: Dashboards & charts  
- `/reports`: Summary insights for stakeholders
